Event Summary

This event is designed to test your code understanding and pattern analysing capabilities. You have to analyse the outputs from the program according to the inputs you give and then guess the algorithm working beneath. Finally, you have to write a program which will exactly replicate the inputs and outputs of the concerned question and submit the solution as a file.

Register before proceeding for the questions, else you will not be evaluated !!!

Nature of Competition

This competition is time bounded (3-hrs), also the evaluation will be on the basis of points according to each question. So a wise approach will be to not waste time stuck on one question.

Input Format

Each question will generate some output on your choice of inputs. If you don't follow the input rules and the output is generated, it will not be correct. Also, the inputs are normal sized, so there is no need to check for very large inputs.

Solution Format

You have to submit your solution code in languages allowed by hackerrank.
